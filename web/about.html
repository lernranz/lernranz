Deutsch

Wo kommen die Daten her?

Die Datenbasis ist die gleiche wie auf dieser Karte, die das Rechenzentrum bereitstellt. Wir wissen also nicht tatsächlich, wie viele Menschen sich in den Räumen aufhalten; sondern nur, wie viele Nutzer in den entsprechenden Eduroam-Access-Points eingebucht sind.
Die Rohdaten finden sich unter *URL*.
Ältere Dumps finden sich auf diesem Server.

Es gibt aber noch andere Lernräume …

Ja, kommt noch. Dazu muss man aber erst die Access-Points ermitteln, die zu dem Raum bzw. Gebäude gehören.

Weitere Lernräume findet man zum Beispiel auf dieser Liste.

Ist das hier die Lernraumampel einer gewissen Liste?

Nein. Das hier ist ein unabhängiges Projekt.

Wie wurde diese Seite umgesetzt?

Ein Cronjob, der Curl und ein Python-Skript zum Parsen verwendet. Dazu ein bisschen HTML und Javascript für die Webseite. Die Webseite ist also rein statisch.

Github

Der Code ist auf Github zu finden.

English

How do you get the data?

We're using the same data as this map, which is provided by the IT Center. So we don't actually know how many people are in the rooms, but we know how many devices are connected to the Eduroam Access Points. The raw data is available at *URL*.
Older dumps can be found on this server.

But these aren't all the rooms!

Yes, there is still work to do. To add more rooms, we have to find out which access points belong to them.

You can find more rooms on this list.

Is this the “Lernraumampel”?

No. This is an independent project.

How was this implemented?

It consists mostly of a cronjob using curl and some Python to parse the data. The web page is just static HTML and Javascript.

Github

The code can be found on Github.
